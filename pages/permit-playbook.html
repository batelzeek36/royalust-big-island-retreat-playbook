Special / Use Permit Playbook — ROYALUST (Big Island)

Land-Use Pathfinding

0) WHICH PERMIT DO WE NEED?
  • If your TMK is in the State Land Use Agricultural/Rural district → apply for a Special Permit (HRS §205-6) via the County Planning Commission.
  • If (less likely) your TMK is in the State Urban district → apply for a County Use Permit (HCC Ch. 25).
Check State district map · GIS layer

1) PREP & PRE-APPLICATION
  • Confirm TMK, State district, County zoning + constraints:
    County maps/GIS
  • Book a pre-application with Hawaiʻi County Planning:
    Apply for a land-use permit

2) PICK THE PATH & GATHER EXHIBITS
  • Special Permit (Ag/Rural) — application form (County Planning Commission):
    Special Permit PDF
  • Use Permit (Urban) — follow HCC Ch.25 procedures (public hearing/standards).
    Code (Ch.25)
  • Exhibits to include (concise):
    • Site plan (parking, fire access) + floor plans; operations plan (max guests, hours, noise).
    • Infrastructure: water, wastewater, toilets; DOH e-Permitting if needed → DOH e-Permitting
    • Traffic/parking memo, neighbor list + outreach plan.
    • Compliance narrative hitting HCC §25-2-65 standards (and HRS §205-6 "unusual & reasonable" test).

3) FILE, NOTICE & HEARING
  • Submit to County Planning per the permit type (Special or Use). The County sets the hearing, publishes notice, and routes agencies for comment.
  • Timeline cue (Use Permit): Public hearing is typically scheduled within ~90 days of acceptance (see Ch.25 procedures).
  • Acreage note: If your Special Permit area is >15 acres, final action goes to the State Land Use Commission after the County hearing.

4) AFTER APPROVAL (CONDITIONS)
  • Read every condition (hours, headcount, parking, sound, tents, sanitation, fire, reporting).
  • Pull any follow-on Plan Approval / Building permits via County:
    Plan Approval application · EPIC help (building permits) · Building permit info
  • If lodging is included, align GET/TAT registrations/filings per your CPA.

5) UNTIL PERMIT IS ISSUED
  • Run day-use only under your PMA; members sleep off-site.
  • Keep private (applications/memberships; no public ticketing), post "Private Event" signage, and maintain a compliance binder on site.
